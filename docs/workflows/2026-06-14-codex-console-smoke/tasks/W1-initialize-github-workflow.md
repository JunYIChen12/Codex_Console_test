# W1 - Initialize GitHub Workflow

## Desired Outcome

Create the smallest useful repository structure for testing a Codex Console development loop through GitHub.

## Scope

- Add repository overview documentation.
- Add agent operating rules.
- Add a pull request template.
- Add a minimal CI workflow.
- Add workflow state and handoff records for this test run.

## Non-Goals

- Do not add product code.
- Do not select a frontend, backend, database, or deployment stack.
- Do not add package dependencies.

## Acceptance Checklist

- [x] GitHub issue exists for this run.
- [x] Files are added on a scoped branch.
- [x] Local verification passes.
- [ ] Pull request links back to the issue.
- [ ] CI passes on the pull request.
- [ ] Workflow result is summarized for the human operator.
